# 🕹️ Flappy Bird: Neon Edition

A high-octane, cyberpunk-themed reimagining of the classic Flappy Bird game. Built with Python and Pygame, this version features vibrant neon aesthetics, dark futuristic backgrounds, and smooth gameplay mechanics.

---

## 🚀 Features
* **Cyberpunk Visuals:** Neon-lit obstacles and atmospheric dark-mode design.
* **Smooth Physics:** Optimized gravity and collision detection for a responsive feel.
* **Dynamic Scoring:** Track your high score as you navigate through the neon pipes.
* **Lightweight:** Minimal dependencies, making it easy to run on any machine.
